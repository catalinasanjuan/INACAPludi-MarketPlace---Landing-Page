# INACAPludi MarketPlace - Landing Page

**INACAPludi MarketPlace** es una aplicación web diseñada para proporcionar una plataforma de compras en línea, enfocada en ofrecer una experiencia de usuario fluida y moderna. Desarrollada con tecnologías web como HTML, CSS y JavaScript, esta aplicación es ideal para pequeñas empresas que buscan digitalizar sus operaciones.

## Características principales

- **Catálogo de productos:** Muestra una lista de productos con imágenes, descripciones y precios.
- **Carrito de compras:** Permite a los usuarios agregar, modificar y eliminar productos del carrito antes de finalizar la compra.
- **Interfaz amigable:** Un diseño atractivo y responsivo que funciona en dispositivos móviles y de escritorio.
- **Estructura escalable:** Código modular para facilitar futuras ampliaciones y personalizaciones.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- `index.html`: Página principal de la aplicación.
- `styles/`: Carpeta que contiene los archivos CSS para el diseño y la presentación.
- `scripts/`: Carpeta con los archivos JavaScript que controlan la funcionalidad de la aplicación.
- `assets/`: Carpeta que almacena imágenes y otros recursos.

## Requisitos del Sistema

Para ejecutar esta aplicación, necesitas:
- Un navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).

## Instrucciones de Uso

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en un navegador.
3. Navega por el catálogo, agrega productos al carrito y experimenta la funcionalidad de la plataforma.
